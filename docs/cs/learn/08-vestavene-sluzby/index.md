# Část 8: Vestavěné služby

Tato sekce pokrývá hotové služby zahrnuté v noex pro běžné potřeby aplikací.

## Kapitoly

### [8.1 EventBus](./01-eventbus.md)

Pub/sub zasílání zpráv:
- Zasílání zpráv podle témat
- Wildcard vzory (`user.*`)
- Případy použití

### [8.2 Cache](./02-cache.md)

In-memory cache:
- LRU eviction
- Podpora TTL
- Limity maximální velikosti

### [8.3 RateLimiter](./03-rate-limiter.md)

Omezování rychlosti:
- Algoritmus posuvného okna
- Limity podle klíče
- API rate limiting

### [8.4 TimerService](./04-timer-service.md)

Trvalé časovače:
- Plánované úlohy
- Persistence
- Opakující se časovače
