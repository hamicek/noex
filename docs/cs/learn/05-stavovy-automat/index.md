# Část 5: Stavový automat

Tato sekce pokrývá GenStateMachine pro explicitní stavy a přechody v komplexní business logice.

## Kapitoly

### [5.1 Kdy použít GenStateMachine](./01-kdy-pouzit.md)

Pochopte, kdy je GenStateMachine správnou volbou:
- Explicitní stavy a přechody
- Komplexní business logika
- Správa workflow

### [5.2 Definice stavů a událostí](./02-definice-stavu.md)

Naučte se základy stavového automatu:
- State handlery
- Zpracování událostí podle stavu
- Timeouty stavů

### [5.3 Workflow objednávky](./03-objednavka-workflow.md)

Praktický příklad:
- Stavy: pending → paid → shipped → delivered
- Události: pay, ship, deliver, cancel
