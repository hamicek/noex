# Tutorials

Step-by-step guides for building real applications with noex.

## Available Tutorials

### [WebSocket Chat Server](./chat-server.md)

Build a real-time chat application with:
- Connection management using GenServers
- Message broadcasting with EventBus
- Supervision for fault tolerance

**Difficulty**: Beginner | **Time**: 30-45 min

---

### [Rate-Limited API](./rate-limited-api.md)

Build a REST API with request throttling:
- Per-user and global rate limiting
- Standard rate limit headers
- Tiered limits for different user types

**Difficulty**: Beginner | **Time**: 20-30 min

---

### [E-commerce Backend](./ecommerce-backend.md)

Build a fault-tolerant backend with:
- Supervision trees for service isolation
- Inter-service communication
- Automatic crash recovery

**Difficulty**: Intermediate | **Time**: 45-60 min

---

### [Monitoring Dashboard](./monitoring-dashboard.md)

Add real-time monitoring to your application:
- Observer for system introspection
- TUI dashboard with blessed
- Web-based monitoring endpoint
- Alert configuration

**Difficulty**: Intermediate | **Time**: 30-45 min

---

## Prerequisites

All tutorials assume:
- Node.js 18+
- Basic TypeScript knowledge
- Familiarity with noex basics (see [Getting Started](../getting-started/index.md))
