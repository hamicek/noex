# Getting Started

Welcome to noex! This section will help you get up and running quickly.

## Prerequisites

- **Node.js 20.0.0** or later
- **TypeScript 5.0+** (recommended)
- Basic understanding of async/await in JavaScript/TypeScript

## Learning Path

### 1. [Installation](./installation.md)
Set up noex in your project with npm, yarn, or pnpm.

### 2. [Quick Start](./quick-start.md)
Create your first GenServer in 5 minutes. Learn the basics of state management and message handling.

### 3. [First Application](./first-application.md)
Build a complete supervised application with multiple services, automatic restarts, and proper shutdown handling.

## What You'll Learn

By the end of this section, you'll understand:

- How to install and configure noex
- The GenServer pattern for stateful services
- How Supervisors provide fault tolerance
- How to use the Registry for service discovery
- Best practices for structuring noex applications

## Coming from Elixir?

If you're familiar with Elixir/OTP, you'll find noex very familiar. Check out the [Elixir Comparison](../concepts/elixir-comparison.md) guide for a quick overview of similarities and differences.
